# Developer Portfolio

A sleek, modern, and highly responsive personal developer portfolio built with the MERN stack (MongoDB, Express, React, Node.js). 

Featuring a premium dark-mode aesthetic with vibrant high-contrast accents, custom glassmorphism UI elements, and scroll-triggered animations.

## 🚀 Features

- **Modern Architecture**: Built with Vite and React for ultra-fast frontend rendering and HMR.
- **Backend Integration**: A robust Express/Node.js backend paired with MongoDB to securely process and store contact form submissions.
- **Premium Design System**: 
  - An ultra-modern dark theme utilizing deep black backgrounds, pure white text, and striking `greenyellow` accents.
  - Implements dynamic CSS hover states, glowing gradients, and translucent glassmorphic frosted-glass panels.
  - Typography powered by the beautifully geometric **Montserrat** font.
- **Fully Responsive**: Architected to fluidly adapt across all device viewports (Desktop, Tablet, Mobile).
- **Project Showcase**: A dedicated portfolio grid detailing full-stack engineering feats, from React Apps to ML Crop models.

## 🛠️ Tech Stack

**Frontend:**
- React (seeded by Vite)
- Vanilla CSS3 (Custom Design System, no frameworks)
- Custom Scalable Vector Graphics (SVGs)

**Backend:**
- Node.js environment
- Express.js REST API
- MongoDB database (via Mongoose)

## 📁 Repository Structure

The project is structured into distinct client and server directories:
- `/frontend` - The React application housing all UI components, assets, and styling logic.
- `/backend` - The Express API housing routing logic, the remote database connection, and data modeling.

## 🚦 Getting Started

### Prerequisites Ready
Make sure you have Node.js (v18+) installed, and a MongoDB instance running.

### 1. Installation
Clone the repository to your local machine:
```bash
git clone https://github.com/Preaah05/portfolio.git
cd portfolio
```

### 2. Backend Setup
Navigate into the backend, install the dependencies, and start the local API:
```bash
cd backend
npm install
# Ensure you create a .env file with your MONGODB_URI and PORT
node server.js
```

### 3. Frontend Setup
Open a new terminal, navigate to the frontend directory, install dependencies, and run the Vite compiler:
```bash
cd frontend
npm install
npm run dev
```

Navigate to `http://localhost:5173` to see the application running.
